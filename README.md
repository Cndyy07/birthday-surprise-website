# 🎂 Birthday Surprise Website

Birthday Surprise Website adalah website ucapan ulang tahun interaktif yang dibuat menggunakan **HTML, CSS, dan JavaScript**. Website ini dirancang dengan tampilan yang menarik, dilengkapi animasi, musik latar, dan berbagai fitur interaktif untuk memberikan pengalaman yang lebih berkesan.

## ✨ Fitur

* 🎉 Tampilan responsif
* 🎨 Background animasi
* 🎵 Musik latar
* 👤 Input nama penerima
* ⌨️ Efek mengetik pada ucapan
* 💌 Pilihan gaya ucapan
* 🎁 Kado virtual interaktif
* 🕯️ Lilin ulang tahun yang dapat ditiup
* 🎊 Efek confetti
* ⬆️ Tombol Back to Top

## 🛠️ Teknologi

* HTML5
* CSS3
* JavaScript
* Lucide Icons

## 📁 Struktur Project

```text
birthday project/
│
├── index.html
├── style.css
├── script.js
├── music.mp3
└── README.md
```

## 🚀 Cara Menjalankan

1. Download atau clone repository ini.
2. Pastikan semua file berada dalam satu folder.
3. Buka file **index.html** di browser.
4. Untuk hasil yang lebih optimal, gunakan **Live Server** di Visual Studio Code.

## 🎨 Kustomisasi

* Ganti musik dengan file `music.mp3` milikmu.
* Ubah isi ucapan pada `script.js`.
* Sesuaikan warna dan tampilan melalui `style.css`.
* Edit struktur halaman pada `index.html`.
